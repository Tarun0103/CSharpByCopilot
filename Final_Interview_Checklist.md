# Final Interview Checklist

### ✅ Core Topics
- OOP concepts with examples (classes, objects, inheritance, polymorphism)
- Abstract vs Interface
- Delegates & Events
- Async / Await
- LINQ (Select, Where, GroupBy, deferred execution)
- Collections (List, Dictionary, HashSet, IEnumerable/ICollection/IList)
- Exception handling (try/catch/finally, throw vs throw ex)
- Memory management (stack vs heap, GC, IDisposable)
- SOLID principles

### ✅ Language & Runtime
- .NET vs .NET Core vs .NET Framework
- Compilation flow (`.cs` → IL → JIT → native)
- `ref` vs `out` vs `in`
- Value type vs Reference type
- Boxing vs Unboxing
- `static` vs `readonly` vs `const`

### ✅ Other Key Concepts
- String vs StringBuilder
- Thread vs Task; Task.Delay vs Thread.Sleep
- Extension methods
- Partial classes
- Reflection and attributes
- Dependency Injection
- Design patterns (Singleton, Factory, Repository)

### ✅ Practice Problems
- Reverse string
- Find duplicates
- Palindrome
- Fibonacci
- LINQ queries
- Sorting

### ✅ Mini Projects
- Student management system (OOP + LINQ + exception handling)
- Employee CRUD console app

---


> Tip: For interviews, talk through your thought process clearly, explain tradeoffs, and be honest about what you don’t know. Being able to reason and reason about alternatives is often more valuable than memorizing exact syntax.
