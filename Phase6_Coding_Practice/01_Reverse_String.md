# 1. Reverse String

## Problem
Given a string, return a new string with the characters in reverse order.

### Approaches
- Use built-in methods (`Array.Reverse`, `Enumerable.Reverse`).
- Manually loop from end to start.

### Example
`"hello" -> "olleh"`

---

## ✅ Interview Prep (Reverse String)
- Discuss time complexity (O(n)) and space complexity.
- Mention immutability of strings and why using `StringBuilder` or `char[]` can help.
