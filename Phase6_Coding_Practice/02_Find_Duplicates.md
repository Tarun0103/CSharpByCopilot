# 2. Find Duplicates

## Problem
Given a list of values, find which values appear more than once.

### Approaches
- Use a `Dictionary<T,int>` or `HashSet<T>` to track seen values.
- Use LINQ grouping.

---

## ✅ Interview Prep (Find Duplicates)
- Explain time complexity: O(n).
- Discuss using `HashSet` for fast membership checks.
