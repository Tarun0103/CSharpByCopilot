# 6. Sorting

## Problem
Sort a collection by one or more fields.

### Approaches
- Use `Array.Sort` or `List<T>.Sort` with a comparison.
- Use LINQ `OrderBy`, `ThenBy`.

---

## ✅ Interview Prep (Sorting)
- Explain the difference between in-place sorting (`Sort`) and creating a new sorted sequence (`OrderBy`).
- Show how to sort by multiple keys.
- Mention complexity: `Sort` is typically O(n log n).
