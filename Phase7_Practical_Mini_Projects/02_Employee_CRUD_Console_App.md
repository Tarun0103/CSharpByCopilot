# 2. Employee CRUD Console App

Build a simple CRUD application for managing employees.

## Requirements
- Create / read / update / delete employees.
- Use unique employee IDs.
- Demonstrate LINQ for search and sorting.

## Concepts used
- OOP (Employee entity, repository)
- Exception handling for input validation
- LINQ (search, ordering)

---

## ✅ Interview Prep (Employee CRUD)
- Explain how you would swap the in-memory repository for a database-backed one.
- Discuss validation and error handling for user input.
- Mention how you might add unit tests for repository logic.
