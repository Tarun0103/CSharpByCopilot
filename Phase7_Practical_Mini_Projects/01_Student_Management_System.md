# 1. Student Management System

Build a simple console app to manage students in memory.

## Requirements
- Add / update / delete students.
- Search students by name or ID.
- List students sorted by name.

## Concepts used
- OOP (Student class, repository)
- LINQ (filtering, sorting)
- Exception handling (validation, invalid input)

---

## ✅ Interview Prep (Student Management)
- Talk about how you would refactor to persist data (e.g., a database).
- Explain how to keep the UI (console) separate from business logic.
- Demonstrate unit testing strategies for the repository logic.
