# C# Interview Prep by Copilot

This workspace is organized into progressive phases to help you master C# and .NET for technical interviews. Each phase contains topic-specific markdown notes and example C# programs.

## Phases

- **Phase 1 – Core C# Fundamentals**: Language basics, data types, control flow, methods, arrays & collections.
- **Phase 2 – OOP (Very Important)**: Classes, encapsulation, inheritance, polymorphism, abstraction, SOLID principles.
- **Phase 3 – Intermediate C#**: Exceptions, delegates & events, LINQ, collections deep dive, generics.
- **Phase 4 – Advanced C#**: Async programming, memory management, file I/O, reflection, attributes.
- **Phase 5 – Real Interview Concepts**: String handling, dependency injection, design patterns.
- **Phase 6 – Coding Practice**: Common algorithmic exercises with C# solutions.
- **Phase 7 – Practical Mini Projects**: Small console apps showing OOP, LINQ, exception handling, etc.

## How to Use

1. Open the folder in VS Code.
2. Browse the phase folders and read the `.md` notes.
3. Run the `.cs` example programs using `dotnet run` or by adding them to a console project.

> Tip: For best practice, create a single solution and multiple projects (e.g., one project per phase) to keep examples isolated and runnable.
